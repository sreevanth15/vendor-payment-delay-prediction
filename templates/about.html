About the Project

Project Overview

The Vendor Payment Delay Prediction System is a comprehensive Data Warehousing and Data Mining (DWDM) project that leverages machine learning to predict vendor payment delays before they occur.

Methodology

Data Processing
  • Dataset: 10,000+ payment records
  • 40+ engineered features
  • Advanced preprocessing techniques
  • SMOTE for class balancing
Feature Engineering
  • Log transformations
  • Categorical encoding (One-Hot)
  • Risk score calculations
  • Temporal features extraction
Machine Learning
  • 4 ML algorithms implemented
  • Novel enhancements in XGBoost & LightGBM
  • Hyperparameter optimization
  • Cross-validation for robustness
Evaluation
  • Accuracy, Precision, Recall, F1-Score
  • AUC-ROC analysis
  • Confusion matrix
  • Business impact analysis

Novel Contributions

Innovations in This Project:
  1. Cost-Sensitive LightGBM: Custom class weights (5x) for delayed payments to prioritize recall
  2. Adaptive XGBoost: Dynamic early stopping and auto class weight calculation
  3. Multi-Metric Evaluation: Simultaneous tracking of AUC-ROC and LogLoss
  4. Business-Focused Features: Custom risk scores combining multiple business factors
